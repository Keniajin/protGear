WELCOME TO protGear

An interactive open source software tool for protein micro-array data management in R.

A flexible generic guide and data-processing suite for protein microarrays in R with statistical functions. Vignette and source code using KILchip v1.0 available here protGear github io/

 WHAT YOU CAN DO WITH protGear:

  • Background correction
  • Coefficient of Variation 
  • Normalisation 
  • Compare normalisation techniques
  • Downstream analysis

This shiny app loads protein microarray data and sample id files.
